# Open a native window pointing to a URL

Open a native window pointing to a URL

## Usage

``` r
rdesk_open_window(
  url,
  title = "RDesk",
  width = 1200,
  height = 800,
  www_path = ""
)
```

## Arguments

- url:

  The target URL to load

- title:

  Window title

- width:

  Window width

- height:

  Window height

- www_path:

  Path to the local assets directory

## Value

A processx process object
