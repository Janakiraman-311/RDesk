# Resolve the www directory for an app

Resolve the www directory for an app

## Usage

``` r
rdesk_resolve_www(www_dir)
```

## Arguments

- www_dir:

  User-provided path to www directory (character) Passing an explicit
  absolute path is the most reliable option and skips the best-effort
  call-stack search.

## Value

Normalized absolute path to a valid www directory
